## 使用语音功能需要bot安装[ffmpeg](https://gitee.com/baihu433/ffmpeg)(小白推荐使用白狐的一键安装)

## cd到崽的根目录，然后↘↓↙
```
git clone https://gitee.com/this_e/xiaokeli.git ./plugins/xiaokeli/
```
---
| 指令 | 说明|
| --------------- | --------------- |
|xx语音|原神星铁角色四国语音|
|xx卡池|原神历代卡池|
|xx攻略|星铁角色攻略,默认关闭|
|塔罗牌|默认关闭，自己开启|
|(自动播报)|米家3游戏的最新视频,设置群号后会自动播报|
|(每日npc委托名)|查询每日委托是否有成就|
|m更新面板|通过米游社获取原神角色面板|
|小可莉帮助|不懂的就发这个|
