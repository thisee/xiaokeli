```

```



---
| 指令 | 说明|
| --------------- | --------------- |
|xx语音|原神星铁角色四国语音|
|xx卡池|原神星铁历代卡池|
|xx攻略|星铁角色攻略,默认关闭|
|塔罗牌|默认关闭，自己开启|
|(自动播报)|米家3游戏的最新视频,设置群号后会自动播报|
|(每日npc委托名)|查询每日委托是否有成就|
|(bilibili解析)|bilibili分享自动解析：卡片，链接(都可)|
|点赞，投币...|对bilibili视频的一些操作|
|小可莉设置|查看设置，具体文件在xiaokeli/config/config.yaml|
|小可莉帮助|不懂的就发这个|
